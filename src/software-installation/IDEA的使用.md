## 快捷键
### 快速格式化代码，
ctrl + shift + L

### 快速生成构造器，
alt + insert

### 查看一个类的层级关系
ctrl + H

### 定位
将光标 放在方法上，输入 ctrl + B，可以选择定位到哪个类的方法
