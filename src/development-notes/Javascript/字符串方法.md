# 字符串方法

## includes() 方法

**介绍：** 查找字符串，如果在*string 中*找到*子字符串*，**includes()** 方法返回*true*。否则，它返回 **false**。

**使用**

```javascript
/** 数组中使用 */
let arr = ['小明', '小杨'];
arr.includes("小明");  // 返回 true
arr.includes("明");  // 返回 false

/** 字符串中使用 */
let arr = ['小明', '小杨'];
arr.includes("小明");  // 返回 true
arr.includes("明");  // 返回 false
```
